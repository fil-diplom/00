# 00 Настройка доступа к GitHub

## Регистрация на GitHub

### 1. Перейти на сайт GitHub

Откройте браузер и перейдите на официальный сайт:  
[https://github.com](https://github.com)

### 2. Нажать "Sign up"

В правом верхнем углу страницы нажмите кнопку **Sign up** (Регистрация).

### 3. Ввести данные для аккаунта

Заполните форму регистрации:
- **Username** — придумайте уникальное имя пользователя.
- **Email address** — укажите ваш рабочий или личный e-mail.
- **Password** — придумайте надёжный пароль.

### 4. Настройка уведомлений

Выберите, хотите ли получать новости и советы от GitHub:
- Отметьте чекбоксы по желанию.

### 5. Подтвердить, что вы не робот

Пройдите проверку **CAPTCHA**, если появится.

### 6. Выбрать план

Выберите план использования GitHub:
- **Free** — бесплатно, подходит для большинства пользователей

### 7. Подтвердить email

- Перейдите на указанный при регистрации e-mail.
- Найдите письмо от GitHub и нажмите кнопку **Verify email address** (Подтвердить адрес электронной почты).

### 8. Настройка профиля (по желанию)

После подтверждения почты можно:
- Добавить фотографию профиля.
- Указать информацию о себе.
- Настроить внешний вид профиля.

### 9. Готово!

Теперь вы можете:
- Создавать репозитории.
- Работать с Git и GitHub.
- Клонировать проекты и делать pull requests.


## Настройка Git и SSH-доступа к GitHub на Windows 10

### 01. Установка Git
1. Откройте браузер и перейдите на сайт: https://git-scm.com/download/win  
2. Сайт автоматически предложит версию для Windows. Нажмите кнопку скачивания.  
3. Скачанный файл будет называться примерно `Git-2.xx.x-64-bit.exe`. Запустите его двойным кликом.  
4. В мастере установки оставьте все параметры по умолчанию (рекомендуется для новичков), нажимая `Next`.  
5. На шаге `Adjusting your PATH environment` выберите вариант "Git from the command line and also from 3rd-party software".  
6. Продолжайте нажимать `Next` и дождитесь завершения установки, затем нажмите `Finish`.  
7. Проверьте установку Git, открыв CMD или PowerShell и введя команду:  
git --version  
Если версия Git отобразилась, значит установка прошла успешно.

### 02. Настройка SSH-доступа

Где работать: в обычном CMD или PowerShell (не обязательно администратором).

### 03. Проверка существующих SSH-ключей

В CMD:
```
dir %userprofile%\.ssh  
```

Если нет файлов `id_rsa` или `id_ed25519`, нужно создать новый ключ.

### 04. Создание нового SSH-ключа

```
ssh-keygen -t ed25519 -C "your_email@example.com"  
```
Замените "your_email@example.com" на email, который используется на GitHub.  
Нажмите Enter для сохранения ключа в стандартном пути.  
Можно задать пароль для ключа или оставить пустым.

### 04. Запуск ssh-agent

```
eval "$(ssh-agent -s)"  
ssh-add ~/.ssh/id_ed25519
```

### 05. Копирование публичного ключа

В CMD:
```
type %userprofile%\.ssh\id_ed25519.pub  
```
Скопируйте в буфер обмена весь текст, начиная с ssh-ed25519 ...

### 06. Добавление ключа на GitHub

Перейдите: https://github.com/settings/ssh/new  
Вставьте скопированный ключ в поле "Key".  
В поле "Title" напишите любое имя (например, My Windows PC).  
Нажмите Add SSH key.

### 06. Проверка подключения

```
ssh -T git@github.com  
```

Должно появиться сообщение:  
"Hi username! You've successfully authenticated, but GitHub does not provide shell access."  
Это значит, что SSH настроен корректно и можно клонировать публичные репозитории через SSH.
